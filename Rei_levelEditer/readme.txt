Q - turns off grid

WASD - Moves Selection box on the right
Scroll - Moves to next Selected box
E - Changes tile mod you can see what it is in the right bar

Left click - places tile
Right Click - pans map

Z - zoom in
X - Zoom out
C - reset

N - New Map enter x and y into cmd (Programe freezes when it waits for you to enter to the CMD)
M - Exports map into the Export folder (This will over write the old one unless you rename it.)
